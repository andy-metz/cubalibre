Description détaillée

crée un client TCP/IP vers un serveur de dessin

envoie 3 requêtes possibles :

ouvrir une fenêtre graphique tracer un segment tracer une ellipse pleine

protocole : Chaque requête est codée sous forme d'une seule String se terminant par le caractère fin de ligne. Sur la ligne, 2 paramètres consécutifs sont séparés par ", ".
